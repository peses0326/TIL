# if ~ elif ~ else 문 연습(2)
### 입력받은 값의 모든 약수 구하고 약수가 2개인 소수를 표기하기
```
x = int(input())
count=0
for a in range(1, x+1):
    if x%a == 0:
        print (f'{a}(은)는 {x}의 약수입니다.')
        count+=1
if count == 2:
    print (f'{x}(은)는 1과 {x}로만 나눌 수 있는 소수입니다.')
```

### 결과
* 입력 7
1(은)는 7의 약수입니다.  
7(은)는 7의 약수입니다.  
7(은)는 1과 7로만 나눌 수 있는 소수입니다.
