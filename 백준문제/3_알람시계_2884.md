if 문으로 전체 시간이 음수가 될 때   
시간에 24시간을 더해 계산하는 조건을 주어서 해결했다  
  

* 풀이  
```
H, M = input().split()
H = int(H)
M = int(M)
fast_time = H * 60 + M - 45
if fast_time < 0:
    fast_time = (H+24) * 60 + M - 45
    fast_time_H = fast_time // 60
    fast_time_M = fast_time % 60
else:
    fast_time_H = fast_time // 60
    fast_time_M = fast_time % 60

print(fast_time_H, fast_time_M)
```
